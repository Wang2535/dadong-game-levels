## 一、奖励系统设计与平衡优化

### 1.1 系统概述

奖励系统是游戏的核心肉鸽(Roguelike)元素，当玩家满足特定条件时，从奖励库中抽取奖励，永久提升角色能力。本系统采用**5级等次架构**，每级包含**30个奖励选项**（15个中性+15个特殊）。

### 1.2 奖励类型分类

| 类型代码 | 类型名称 | 说明 | 示例 |
|----------|----------|------|------|
| TYPE_A | 零消耗卡牌 | 获得可零消耗使用的卡牌 | 零消耗史诗卡 |
| TYPE_B | 额外回合 | 获得额外行动回合 | 额外行动阶段 |
| TYPE_C | 阶段扩展 | 扩展特定阶段的功能 | 扩展行动阶段 |
| TYPE_D | 即时资源 | 立即获得资源 | 立即获得5点算力 |
| TYPE_E | 资源上限 | 永久提升资源上限 | 算力上限+3 |
| TYPE_F | 资源恢复 | 提升每回合资源恢复 | 每回合资金+2 |
| TYPE_G | 抽卡优化 | 提升抽卡效率或品质 | 抽卡时品质+1 |
| TYPE_H | 判定增强 | 增强判定机制效果 | 判定难度-1 |
| TYPE_I | 响应强化 | 增强响应阶段能力 | 响应次数+1 |
| TYPE_J | 特殊能力 | 获得独特游戏机制 | 资源转换能力 |

### 1.3 奖励库结构设计

#### 1.3.1 奖励分类配比

| 分类 | 数量 | 说明 | 抽取规则 |
|------|------|------|----------|
| 中性奖励 | 15个 | 攻击方与防御方通用 | 第1抽必从中性奖励中抽取 |
| 特殊奖励 | 15个 | 攻击方或防御方专属 | 第2抽必从特殊奖励中抽取 |
| **合计** | **30个** | - | 第3抽从剩余28个中抽取 |

#### 1.3.2 抽取机制规则

```
抽取流程：
┌─────────────────────────────────────────────────────────────┐
│  第1抽：从15个中性奖励中随机抽取1个                          │
│       ↓                                                      │
│  第2抽：从15个特殊奖励中随机抽取1个                          │
│       ↓                                                      │
│  第3抽：从剩余28个奖励（30-2）中随机抽取1个                   │
│       ↓                                                      │
│  玩家选择：从3个奖励中选择1个获得                             │
└─────────────────────────────────────────────────────────────┘
```

## 二、胜利条件系统重构（血量机制）

### 2.1 重构原则

1. **血量化设计**：渗透等级和安全等级作为双方的血量系统
2. **独立计算**：胜利条件独立于奖励系统，直接由血量数值决定
3. **平衡对抗**：攻防双方血量上限相同（75级），确保公平性
4. **清晰目标**：胜利条件明确，玩家目标清晰

### 2.2 血量系统核心机制

#### 2.2.1 渗透等级系统（攻击方血量）

```

初始值：0

失败阈值：被防御方降至0（威胁清除）

提升方式：
- 成功攻击：+1至+3
- 控制区域：+1/回合
- 特殊攻击卡牌：+2至+5
- 奖励效果：+0.5至+3/回合

降低方式：
- 防御方成功防御：-1至-2
- 防御方反击：-2至-5
- 特殊防御卡牌：-3至-5
- 失去区域控制：-2/回合
```

### 2.4 速攻/速防胜利条件详解

#### 2.4.1 速攻胜利（攻击方）

**达成条件**（必须同时满足）：
1. 在第8回合结束前，渗透等级达到50级
2. 防御方安全等级在第8回合结束前未达到50级

**设计意图**：
- 鼓励攻击方快速进攻，但需保持领先优势
- 防止双方同时达成速攻条件导致的平局
- 增加策略深度：攻击方需在进攻与压制之间权衡

**失败情况**：
- 第8回合结束时渗透等级<50：速攻失败
- 第8回合结束时双方均≥50：速攻失败，游戏继续
- 第8回合结束时防御方≥50但攻击方<50：防御方有机会达成速防

#### 2.4.2 速防胜利（防御方）

**达成条件**（必须同时满足）：
1. 在第8回合结束前，安全等级达到50级
2. 攻击方渗透等级在第8回合结束前未达到50级

**设计意图**：
- 鼓励防御方快速建立防御体系
- 与速攻形成对抗关系，增加前期博弈
- 若双方均快速发展，则进入中后期对局
