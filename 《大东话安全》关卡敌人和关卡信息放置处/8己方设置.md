# 第八关 己方设置

## 一、关卡8解锁卡牌

### 1.1 解锁条件
通关第八关"密码之战"后解锁

### 1.2 解锁卡牌列表

#### 卡牌1：多因素认证

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LI8-1T2 |
| **卡牌名称** | 多因素认证 |
| **来源类** | L（关卡卡牌） |
| **功能类** | I（即时判定类） |
| **科技等级** | T2 |
| **稀有度** | 稀有 |
| **行动点消耗** | 2 |
| **效果** | 本回合内，所有友方标记免疫"被解密"效果。如果密码破解类敌人尝试使用"密码分析"技能，该技能反噬，敌人失去1个标记。 |
| **原文依据** | 多因素认证大大增加了密码破解的难度，是现代安全体系的重要组成部分。 |
| **设计说明** | 基于密码学中多因素认证的概念设计，让玩家体验多层认证的安全效果 |

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LF8-1T3 |
| **卡牌名称** | 密钥更新 |
| **来源类** | L（关卡卡牌） |
| **功能类** | F（基础数值类） |
| **科技等级** | T3 |

| **设计说明** | 基于原文中密码机制需要不断更新的概念设计，让玩家体验密钥更新的安全效果 |

## 二、卡牌加入时机

### 2.1 关卡8解锁卡牌
- 通关第八关后立即加入玩家卡池
- 在后续关卡中可供玩家使用
- 与关卡1-7解锁卡牌叠加

---

## 三、策略建议

### 3.1 关卡8解锁卡牌使用建议
1. **多因素认证**：在密码破解类敌人准备使用"密码分析"时使用，可以让攻击反噬
2. **密钥更新**：当多个区域被解密时使用，可以一次性恢复所有区域

### 3.2 与之前卡牌配合
- 配合**隐私安全意识(LF7-2T3)**使用，可以同时保护隐私和密码安全
- 配合**组件化防御(LF4-1T2)**使用，可以形成更完善的防御体系
