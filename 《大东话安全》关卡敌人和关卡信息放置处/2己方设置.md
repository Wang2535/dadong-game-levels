# 第二关 己方设置

## 一、关卡2解锁卡牌

### 1.1 解锁条件
通关第二关"AI的抉择"后解锁

### 1.2 解锁卡牌列表

#### 卡牌1：人工干预

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LI2-1T1 |
| **卡牌名称** | 人工干预 |
| **来源类** | L（关卡卡牌） |
| **功能类** | I（即时判定类） |
| **科技等级** | T1 |
| **稀有度** | 稀有 |
| **行动点消耗** | 2 |
| **效果** | 本回合内，AI类敌人的"计算否定"技能无效。友方角色在AI类敌人所在区域放置标记时行动点消耗-1。 |
| **原文依据** | "飞行员可能需要马上关闭失速自动保护系统，才能获取飞机的完全控制权。" |
| **设计说明** | 基于原文中人工干预夺回控制权的概念设计，让玩家可以突破AI的判断限制 |

#### 卡牌2：行为异常检测

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LI2-2T2 |
| **卡牌名称** | 行为异常检测 |
| **来源类** | L（关卡卡牌） |
| **功能类** | I（即时判定类） |
| **科技等级** | T2 |
| **稀有度** | 史诗 |
| **行动点消耗** | 2 |
| **效果** | 本回合内，AI攻击者的所有自动技能失效。如果AI攻击者尝试使用主动技能，该技能反噬，AI攻击者失去2个标记。 |
| **原文依据** | "如果一名工作人员通常在北京工作，突然有一天早上从纽约登录，这是一种反常现象——人工智能可以看出这是一种反常现象。" |
| **设计说明** | 基于原文中行为异常检测的概念设计，利用AI的检测能力反制AI攻击 |

## 二、卡牌加入时机

### 2.1 关卡2解锁卡牌
- 通关第二关后立即加入玩家卡池
- 在后续关卡中可供玩家使用
- 与关卡1解锁卡牌叠加

---

## 三、策略建议

### 3.1 关卡2解锁卡牌使用建议
1. **人工干预**：在AI类敌人准备使用"计算否定"时使用，可以打破AI的控制
2. **行为异常检测**：当AI攻击者频繁使用自动技能时使用，可以让攻击反噬

### 3.2 与初始卡牌配合
- 配合**入侵检测(NI0-1T1)**使用，可以更早发现AI类敌人的攻击意图
- 配合**行为分析(NI0-5T2)**使用，可以更准确地判断AI类敌人的行为模式
