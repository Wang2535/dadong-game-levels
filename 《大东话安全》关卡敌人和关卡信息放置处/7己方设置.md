# 第七关 己方设置

## 一、关卡7解锁卡牌

### 1.1 解锁条件
通关第七关"隐私透明"后解锁

### 1.2 解锁卡牌列表

#### 卡牌1：MAC地址随机化

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LF7-1T2 |
| **卡牌名称** | MAC地址随机化 |
| **来源类** | L（关卡卡牌） |
| **功能类** | F（基础数值类） |
| **科技等级** | T2 |
| **稀有度** | 稀有 |
| **行动点消耗** | 2 |
| **效果** | 为目标区域添加"隐私保护"。探针盒子在该区域无法获得"信息碎片"，且探针盒子的标记在该区域效果减半。持续3回合。 |
| **原文依据** | "2014年，苹果在iOS 8中加入了一个旨在保护用户隐私的新功能'MAC地址随机化'……从而帮助保护用户隐私，防止基于通过设备MAC地址进行用户追踪。" |
| **设计说明** | 基于原文中MAC地址随机化技术设计，让玩家体验隐私保护技术的作用 |

#### 卡牌2：隐私安全意识

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LF7-2T3 |
| **卡牌名称** | 隐私安全意识 |
| **来源类** | L（关卡卡牌） |
| **功能类** | F（基础数值类） |
| **科技等级** | T3 |
| **稀有度** | 史诗 |
| **行动点消耗** | 3 |
| **效果** | 所有友方角色获得"隐私保护"状态，持续2回合。在此期间，隐私窃取类敌人的所有技能对友方无效。隐私窃贼失去所有"隐私卡"。 |
| **原文依据** | "作为一个普通用户，更重要的还是隐私安全意识的加强，从自己身上斩断不良厂商伸来的黑手。" |
| **设计说明** | 基于原文中隐私安全意识的重要性设计，让玩家体验意识在隐私保护中的关键作用 |

## 二、卡牌加入时机

### 2.1 关卡7解锁卡牌
- 通关第七关后立即加入玩家卡池
- 在后续关卡中可供玩家使用
- 与关卡1-6解锁卡牌叠加

---

## 三、策略建议

### 3.1 关卡7解锁卡牌使用建议
1. **MAC地址随机化**：在公共场所区域使用，防止被探针盒子追踪
2. **隐私安全意识**：当隐私窃贼获得较多"隐私卡"时使用，可以一次性清除威胁

### 3.2 与之前卡牌配合
- 配合**安全意识觉醒(LF3-1T2)**使用，可以同时防御病毒和隐私窃取
- 配合**行为异常检测(LI2-2T2)**使用，可以更早发现隐私窃取行为
