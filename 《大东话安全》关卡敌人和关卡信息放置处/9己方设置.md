# 第九关 己方设置

## 一、关卡9解锁卡牌

### 1.1 解锁条件
通关第九关"账号保卫战"后解锁

### 1.2 解锁卡牌列表

#### 卡牌1：异常行为检测

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LI9-1T2 |
| **卡牌名称** | 异常行为检测 |
| **来源类** | L（关卡卡牌） |
| **功能类** | I（即时判定类） |
| **科技等级** | T2 |
| **稀有度** | 稀有 |
| **行动点消耗** | 2 |
| **效果** | 揭示盗号黑手设置的所有"钓鱼陷阱"，并使其失效。每揭示一个陷阱，友方获得1行动点。 |
| **原文依据** | "移动互联网企业可以通过大数据技术的方式分析用户行为，进行用户画像，通过建立用户黑名单的方式，对高危标记账号做行径追踪，及时制止恶意行为。" |
| **设计说明** | 基于原文中异常行为检测的概念设计，让玩家可以发现并清除钓鱼陷阱 |

#### 卡牌2：设备指纹识别

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LI9-2T3 |
| **卡牌名称** | 设备指纹识别 |
| **来源类** | L（关卡卡牌） |
| **功能类** | I（即时判定类） |
| **科技等级** | T3 |
| **稀有度** | 史诗 |
| **行动点消耗** | 3 |
| **效果** | 移除做号集团在指定区域的所有标记，并解除该区域的"黑产运营"状态。做号集团的"批量产号"技能下回合无效。 |
| **原文依据** | "企业可以通过价格相对高昂的移动设备信息入手，用设备可信ID的唯一性作为判断账号是否真实唯一的重要指标，从而识别和限制小号复用行为。" |
| **设计说明** | 基于原文中设备指纹识别技术设计，让玩家可以识别和清除虚假账号 |

## 二、卡牌加入时机

### 2.1 关卡9解锁卡牌
- 通关第九关后立即加入玩家卡池
- 在后续关卡中可供玩家使用
- 与关卡1-8解锁卡牌叠加

---

## 三、策略建议

### 3.1 关卡9解锁卡牌使用建议
1. **异常行为检测**：在账号中心等关键区域使用，可以发现隐藏的钓鱼陷阱
2. **设备指纹识别**：当做号集团大量产号时使用，可以一次性清除批量账号

### 3.2 与之前卡牌配合
- 配合**多因素认证(LI8-1T2)**使用，可以同时保护账号和密码安全
- 配合**密钥更新(LF8-1T3)**使用，可以形成完善的账号安全体系
