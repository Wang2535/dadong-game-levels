# 第五关 己方设置

## 一、关卡5解锁卡牌

### 1.1 解锁条件
通关第五关"致命缺陷"后解锁

### 1.2 解锁卡牌列表

#### 卡牌1：冗余设计

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LF5-1T2 |
| **卡牌名称** | 冗余设计 |
| **来源类** | L（关卡卡牌） |
| **功能类** | F（基础数值类） |
| **科技等级** | T2 |
| **稀有度** | 稀有 |
| **行动点消耗** | 2 |
| **效果** | 为目标区域添加"冗余保护"。该区域友方标记数量不会低于1，且设计缺陷类敌人的"故障模式触发"技能对该区域无效。持续3回合。 |
| **原文依据** | "波音可能会对其737 MAX上失速自动保护系统进行系统更新时，解决传感器安全余度和数据真实性验证的设计缺陷。" |
| **设计说明** | 基于原文中安全余度的概念设计，通过冗余设计防止单点故障 |

#### 卡牌2：安全培训

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LF5-2T2 |
| **卡牌名称** | 安全培训 |
| **来源类** | L（关卡卡牌） |
| **功能类** | F（基础数值类） |
| **科技等级** | T2 |
| **稀有度** | 史诗 |
| **行动点消耗** | 3 |
| **效果** | 所有友方角色获得"安全意识"状态，持续2回合。在此期间，设计缺陷类敌人的所有技能对友方无效。 |
| **原文依据** | "平时对飞行员应该多加培训，让他们有处理这种风险的准备。" |
| **设计说明** | 基于原文中安全培训的重要性设计，让玩家体验培训在风险应对中的作用 |

## 二、卡牌加入时机

### 2.1 关卡5解锁卡牌
- 通关第五关后立即加入玩家卡池
- 在后续关卡中可供玩家使用
- 与关卡1-4解锁卡牌叠加

---

## 三、策略建议

### 3.1 关卡5解锁卡牌使用建议
1. **冗余设计**：在关键区域部署，防止因设计缺陷导致的全面崩溃
2. **安全培训**：当设计缺陷类敌人频繁攻击时使用，可以全面免疫

### 3.2 与之前卡牌配合
- 配合**组件化防御(LF4-1T2)**使用，可以形成更完善的防御体系
- 配合**人工干预(LI2-1T1)**使用，在自动化系统失效时由人工接管
