# 第四关 己方设置

## 一、关卡4解锁卡牌

### 1.1 解锁条件
通关第四关"组件化攻击"后解锁

### 1.2 解锁卡牌列表

#### 卡牌1：火焰检测工具

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LI4-1T2 |
| **卡牌名称** | 火焰检测工具 |
| **来源类** | L（关卡卡牌） |
| **功能类** | I（即时判定类） |
| **科技等级** | T2 |
| **稀有度** | 稀有 |
| **行动点消耗** | 2 |
| **效果** | 揭示火焰病毒在所有区域的隐藏标记，并移除其中一半（向上取整）。 |

| **设计说明** | 基于原文中火焰病毒检测工具的概念设计，让玩家可以发现并清除隐藏的病毒 |

## 二、卡牌加入时机

### 2.1 关卡4解锁卡牌
- 通关第四关后立即加入玩家卡池
- 在后续关卡中可供玩家使用
- 与关卡1-3解锁卡牌叠加

## 三、策略建议

### 3.1 关卡4解锁卡牌使用建议
1. **火焰检测工具**：当怀疑有隐藏标记时使用，可以一次性清除多个隐藏威胁
2. **组件化防御**：在面对组件化攻击时使用，可以全面防护

### 3.2 与之前卡牌配合
- 配合**网络隔离(LI3-1T1)**使用，先检测后隔离，效果更佳
- 配合**安全意识觉醒(LF3-1T2)**使用，可以形成多层防御体系
