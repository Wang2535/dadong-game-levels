# 第一关 己方设置

## 一、初始卡牌选择

### 1.1 初始卡牌说明
初始卡牌是指玩家在开始游戏、尚未挑战任何关卡时就能使用的卡牌。这些卡牌从T1卡池中选取，适合新手玩家使用。

### 1.2 初始卡牌列表

| 序号 | 卡牌编号 | 卡牌名称 | 科技等级 | 稀有度 | 选择理由 |
|------|----------|----------|----------|--------|----------|
| 1 | NF0-1T1 | 防火墙部署 | T1 | 普通 | 最基础的防御卡牌，效果简单直接，新手容易理解 |
| 2 | NI0-1T1 | 入侵检测 | T1 | 普通 | 核心防御机制，帮助新手了解入侵检测概念 |
| 3 | NF0-4T1 | 补丁管理 | T1 | 普通 | 简单的数值增益，适合新手熟悉游戏节奏 |
| 4 | NF0-6T1 | 备份恢复 | T1 | 普通 | 提供恢复能力，给新手容错空间 |
| 5 | NI0-5T2 | 行为分析 | T2 | 稀有 | T2入门卡牌，效果直观易懂 |
| 6 | NI0-2T1 | 日志监控 | T1 | 普通 | 配合入侵检测使用，帮助新手建立分析思维 |
| 7 | NE0-2T1 | 紧急补给 | T1 | 普通 | 通用卡牌，帮助新手应对困难情况 |
| 8 | NE0-1T1 | 资源调配 | T1 | 普通 | 提供资源管理灵活性，适合新手学习资源规划 |

## 二、关卡1解锁卡牌

### 2.1 解锁条件
通关第一关"病毒初现"后解锁

### 2.2 解锁卡牌列表

#### 卡牌1：签名接种

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LF1-1T1 |
| **卡牌名称** | 签名接种 |
| **来源类** | L（关卡卡牌） |
| **功能类** | F（基础数值类） |
| **科技等级** | T1 |
| **稀有度** | 稀有 |
| **行动点消耗** | 2 |
| **效果** | 为目标区域添加"已接种"标记。病毒类敌人的感染技能对该区域无效，持续3回合。 |
| **原文依据** | "一旦Elk Cloner被删除，以前感染的磁盘将不会被重新感染，因为它已经在其目录中包含Elk Cloner'签名'。也可以将'签名'写入磁盘来'接种'未受感染的磁盘对抗Elk Cloner。" |
| **设计说明** | 基于原文中"签名接种"的概念设计，让玩家体验病毒免疫机制 |

#### 卡牌2：系统重写

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LI1-1T1 |
| **卡牌名称** | 系统重写 |
| **来源类** | L（关卡卡牌） |
| **功能类** | I（即时判定类） |
| **科技等级** | T1 |
| **稀有度** | 稀有 |
| **行动点消耗** | 3 |
| **效果** | 移除指定病毒类敌人在目标区域的所有标记。需要弃置一张手牌。 |
| **原文依据** | "可以使用Apple的MASTER CREATE实用程序或其他实用程序删除病毒，以将新的DOS副本重新写入受感染的磁盘。" |
| **设计说明** | 基于原文中"系统重写"清除病毒的方法设计，作为强力的病毒清除手段 |

## 三、卡牌加入时机

### 3.1 初始卡牌
- 游戏开始时自动加入玩家卡池
- 无需任何解锁条件

### 3.2 关卡1解锁卡牌
- 通关第一关后立即加入玩家卡池
- 在后续关卡中可供玩家使用

---

## 四、策略建议

### 4.1 初始卡牌使用建议
1. **防火墙部署**：优先在核心区域部署，建立基础防线
2. **入侵检测**：配合防火墙使用，提高检测效率
3. **补丁管理**：在敌人攻击后使用，快速恢复防御能力
4. **备份恢复**：保留作为应急手段，关键时刻使用

### 4.2 关卡1解锁卡牌使用建议
1. **签名接种**：在病毒扩散前使用，预防感染效果更好
2. **系统重写**：当病毒标记较多时使用，可以一次性清除
