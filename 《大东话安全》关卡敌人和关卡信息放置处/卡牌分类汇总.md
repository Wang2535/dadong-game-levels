# 卡牌分类汇总

## 一、卡牌分类系统说明

### 1.1 第一大类型：卡牌来源类

| 分类代码 | 分类名称 | 说明 |
|----------|----------|------|
| S | 角色特殊卡牌类 | 角色专属的特殊能力卡牌 |
| N | 正常卡牌类 | 常规游戏卡牌，从卡池中抽取 |
| L | 关卡卡牌类 | 通过关卡解锁获得的卡牌 |

### 1.2 第二大类型：功能类型

| 分类代码 | 分类名称 | 说明 |
|----------|----------|------|
| E | 资源交换类 | 主要功能为交换或获得资源 |
| F | 基础数值类 | 主要功能为基础的增减数值 |
| I | 即时判定类 | 主要功能为具有即时判定效果 |
| A | 延时判定类 | 主要功能为具有延时判定效果 |
| D | 数值降低类 | 主要功能为降低敌方数值（如资源、行动点等） |
| O | 其他类卡牌 | 无法归入上述功能类型的卡牌 |

### 1.3 卡牌编号规则

**所有卡牌统一采用 XZYa-bC 格式**
- X = 来源类代码 (S/N/L)
- Z = 功能类代码 (E/F/I/A/D/O)
- a = 关卡数 (现有卡牌为0，关卡解锁卡牌为对应关卡数)
- b = 该卡牌序号
- C = 科技树等级 (T0-T5)

**示例**：
- NF0-1T1 表示：正常卡牌、基础数值类、现有卡牌第1张、T1等级
- LE1-1T1 表示：关卡卡牌、资源交换类、第1关第1张、T1等级
