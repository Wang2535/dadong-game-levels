# 第三关 己方设置

## 一、关卡3解锁卡牌

### 1.1 解锁条件
通关第三关"蠕虫危机"后解锁

### 1.2 解锁卡牌列表

#### 卡牌1：网络隔离

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LI3-1T1 |
| **卡牌名称** | 网络隔离 |
| **来源类** | L（关卡卡牌） |
| **功能类** | I（即时判定类） |
| **科技等级** | T1 |
| **稀有度** | 稀有 |
| **行动点消耗** | 2 |
| **效果** | 选择一个区域进行网络隔离。该区域本回合内不受任何蠕虫病毒的复制和传播影响。蠕虫类敌人的"自我繁殖"技能对该区域无效。 |
| **原文依据** | "一旦发现蠕虫，要在尽量短的时间内对其进行响应……通过防火墙、路由器、或者HIDS的互动将感染了蠕虫的主机隔离。" |
| **设计说明** | 基于原文中网络隔离的防御方法设计，阻止蠕虫病毒的扩散 |

#### 卡牌2：安全意识觉醒

| 属性 | 内容 |
|------|------|
| **卡牌编号** | LF3-1T2 |
| **卡牌名称** | 安全意识觉醒 |
| **来源类** | L（关卡卡牌） |
| **功能类** | F（基础数值类） |
| **科技等级** | T2 |
| **稀有度** | 史诗 |
| **行动点消耗** | 3 |
| **效果** | 所有友方角色获得"安全意识"状态，持续3回合。在此期间，蠕虫病毒的感染技能对友方标记无效。 |
| **原文依据** | "最最重要的还是安全意识，不要因为好奇而造成祸患。网络千万条，安全第一条……绝不浏览暗网和不良站点！" |
| **设计说明** | 基于原文中安全意识的重要性设计，让玩家体验安全意识在防御中的关键作用 |

## 三、策略建议

### 3.1 关卡3解锁卡牌使用建议
1. **网络隔离**：在蠕虫病毒开始扩散时使用，阻止其向其他区域传播
2. **安全意识觉醒**：作为终极防御手段，可以全面免疫蠕虫感染

### 3.2 与之前卡牌配合
- 配合**签名接种(LF1-1T1)**使用，可以同时防御病毒和蠕虫
- 配合**系统重写(LI1-1T1)**使用，先清除后隔离，效果更佳
