## 一、等级系统核心机制重构

### 1.1 系统概述

**渗透等级**（攻击方）与**安全等级**（防御方）作为双方的核心血量系统，其增长与降低完全由卡牌效果驱动，替代原有的自动增长机制。

### 1.2 等级系统基础参数

| 参数 | 攻击方（渗透等级） | 防御方（安全等级） |
|------|-------------------|-------------------|
| **范围** | 0-75 | 0-75 |
| **初始值** | 0 | 0 |
| **胜利阈值** | 75（完全渗透） | 75（绝对安全） |
| **增长方式** | 卡牌效果直接提升 | 卡牌效果直接提升 |
| **降低方式** | 对方卡牌效果降低 | 对方卡牌效果降低 |

## 二、卡牌效果与等级系统绑定

### 2.1 设计原则

1. **攻击方偏向**：更多卡牌具有降低对方安全等级的效果
2. **防御方偏向**：更多卡牌具有提升自身安全等级的效果
3. **完全绑定**：所有等级变化必须通过卡牌效果触发，无自动增长
4. **策略深度**：卡牌效果设计需考虑风险收益比

## 三、胜利条件系统

### 3.1 胜利条件类型

#### 3.1.1 攻击方胜利条件

| 胜利条件 | 达成要求 | 策略类型 |
|----------|----------|----------|
| **完全渗透** | 渗透等级达到75 | 核心目标 |
| **安全瓦解** | 使防御方安全等级降至0 | 压制策略 |
| **速攻胜利** | 第8回合前渗透等级达到50 **且** 防御方安全等级<50 | 速攻策略 |
| **持久胜利** | 第20回合后比较双方等级，渗透等级>安全等级 | 持久策略 |

#### 3.1.2 防御方胜利条件

| 胜利条件 | 达成要求 | 策略类型 |
|----------|----------|----------|
| **绝对安全** | 安全等级达到75 | 核心目标 |
| **威胁清除** | 使攻击方渗透等级降至0 | 清剿策略 |
| **速防胜利** | 第8回合前安全等级达到50 **且** 攻击方渗透等级<50 | 速防策略 |
| **持久胜利** | 第20回合后比较双方等级，安全等级>渗透等级 | 持久策略 |

### 3.2 持久胜利条件详解（第20回合强制判定）

#### 3.2.1 判定规则

```
第20回合结束时的判定流程：

┌─────────────────────────────────────────────────────────────┐
│  1. 比较攻击方渗透等级与防御方安全等级                         │
│       ↓                                                      │
│  2. 若渗透等级 > 安全等级 → 攻击方胜利（持久胜利）            │
│       ↓                                                      │
│  3. 若安全等级 > 渗透等级 → 防御方胜利（持久胜利）            │
│       ↓                                                      │
│  4. 若渗透等级 = 安全等级 → 进入加时赛（第21回合开始）         │
│       ↓                                                      │
│  5. 加时赛中，先提升等级者获胜                                │
└─────────────────────────────────────────────────────────────┘
```

#### 3.2.2 设计意图

- **避免无限对局**：20回合后强制结束游戏
- **鼓励主动进攻**：被动防守无法在持久战中获胜
- **平衡双方机会**：等级高者获胜，体现对局优势
- **加时赛机制**：平局时进入紧张刺激的抢分阶段

#### 3.2.3 数值参考

| 对局类型 | 预计20回合时等级 | 胜利方 |
|----------|------------------|--------|
| 攻击方优势 | 渗透55 vs 安全40 | 攻击方 |
| 防御方优势 | 渗透40 vs 安全55 | 防御方 |
| 均衡对局 | 渗透45 vs 安全45 | 加时赛 |
| 速攻失败后续 | 渗透35 vs 安全50 | 防御方 |

### 3.3 速攻/速防胜利条件

#### 3.3.1 速攻胜利（攻击方）

**达成条件**：
1. 在第8回合结束前，渗透等级达到50级
2. 防御方安全等级在第8回合结束前未达到50级

**失败情况**：
- 第8回合结束时渗透等级<50：速攻失败
- 第8回合结束时双方均≥50：速攻失败，游戏继续
- 第8回合结束时防御方≥50但攻击方<50：防御方有机会达成速防

#### 3.3.2 速防胜利（防御方）

**达成条件**：
1. 在第8回合结束前，安全等级达到50级
2. 攻击方渗透等级在第8回合结束前未达到50级
