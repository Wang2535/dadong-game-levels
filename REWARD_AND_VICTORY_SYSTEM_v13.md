## 一、奖励系统设计与平衡优化

### 1.1 系统概述

奖励系统是游戏的核心肉鸽(Roguelike)元素，当玩家满足特定条件时，从奖励库中抽取奖励，永久提升角色能力。本系统采用**5级等次架构**，每级包含**30个奖励选项**（15个中性+15个特殊）。

### 1.2 奖励类型分类

| 类型代码 | 类型名称 | 说明 | 示例 |
|----------|----------|------|------|
| TYPE_A | 零消耗卡牌 | 获得可零消耗使用的卡牌 | 零消耗史诗卡 |
| TYPE_B | 额外回合 | 获得额外行动回合 | 额外行动阶段 |
| TYPE_C | 阶段扩展 | 扩展特定阶段的功能 | 扩展行动阶段 |
| TYPE_D | 即时资源 | 立即获得资源 | 立即获得5点算力 |
| TYPE_E | 资源上限 | 永久提升资源上限 | 算力上限+3 |
| TYPE_F | 资源恢复 | 提升每回合资源恢复 | 每回合资金+2 |
| TYPE_G | 抽卡优化 | 提升抽卡效率或品质 | 抽卡时品质+1 |
| TYPE_H | 判定增强 | 增强判定机制效果 | 判定难度-1 |
| TYPE_I | 响应强化 | 增强响应阶段能力 | 响应次数+1 |
| TYPE_J | 特殊能力 | 获得独特游戏机制 | 资源转换能力 |

### 1.3 奖励库结构设计

#### 1.3.1 奖励分类配比

| 分类 | 数量 | 说明 | 抽取规则 |
|------|------|------|----------|
| 中性奖励 | 15个 | 攻击方与防御方通用 | 第1抽必从中性奖励中抽取 |
| 特殊奖励 | 15个 | 攻击方或防御方专属 | 第2抽必从特殊奖励中抽取 |
| **合计** | **30个** | - | 第3抽从剩余28个中抽取 |

#### 1.3.2 抽取机制规则

```
抽取流程：
┌─────────────────────────────────────────────────────────────┐
│  第1抽：从15个中性奖励中随机抽取1个                          │
│       ↓                                                      │
│  第2抽：从15个特殊奖励中随机抽取1个                          │
│       ↓                                                      │
│  第3抽：从剩余28个奖励（30-2）中随机抽取1个                   │
│       ↓                                                      │
│  玩家选择：从3个奖励中选择1个获得                             │
└─────────────────────────────────────────────────────────────┘
```

       P(中性) = 14/28 = 50% (若前2抽各抽1个不同类型)
       P(特殊-攻击方) = 7-8/28 ≈ 25-29%
       P(特殊-防御方) = 6-7/28 ≈ 21-25%

## 二、胜利条件系统重构

### 2.1 重构原则

1. **独立性**：胜利条件不再与科技树、奖励系统直接绑定
2. **平衡性**：攻防双方胜利难度相当
3. **可玩性**：提供多样化的胜利路径
4. **清晰性**：胜利条件明确易懂

### 2.2 胜利条件类型

#### 2.2.1 攻击方胜利条件

| 胜利条件 | 达成要求 | 难度评级 | 策略类型 |
|----------|----------|----------|----------|
| **完全渗透** | 渗透等级达到10 | ★★★★☆ | 核心目标 |
| **区域控制** | 控制全部4个区域 | ★★★★★ | 控制策略 |
| **资源枯竭** | 使防御方所有资源降至0 | ★★★★☆ | 消耗策略 |
| **速攻胜利** | 在第8回合前渗透等级达到8 | ★★★★★ | 速攻策略 |
| **持久胜利** | 游戏进行20回合后渗透等级≥8 | ★★★☆☆ | 持久策略 |

#### 2.2.2 防御方胜利条件

| 胜利条件 | 达成要求 | 难度评级 | 策略类型 |
|----------|----------|----------|----------|

### 2.3 核心胜利机制

#### 2.3.1 渗透等级系统（攻击方）

```
渗透等级范围：0-10
初始值：0
胜利阈值：10

提升方式：
- 成功攻击：+1
- 控制区域：+0.5/回合
- 特殊卡牌：+1至+3
- 奖励效果：+0.5至+3/回合

降低方式：
- 防御方反击：-1
- 失去区域控制：-0.5/回合
- 特殊防御卡牌：-1至-2
```

#### 2.3.2 安全等级系统（防御方）

```
安全等级范围：0-10
初始值：5
胜利阈值：10
失败阈值：0

- 控制区域：+0.5/回合
- 特殊防御卡牌：+1至+3

降低方式：
- 遭受攻击：-1

```

// 胜利条件配置（独立于奖励系统）
interface VictoryCondition {
  type: 'infiltration' | 'security' | 'control' | 'resource' | 'speed' | 'endurance';
  targetValue: number;
  currentValue: number;

  applyEffect(player: Player): void;
}
