## 游戏简介

《道高一丈：数字博弈》是一款以网络安全攻防为主题的策略卡牌游戏。玩家分为**进攻方**和**防御方**，通过使用各种技术卡牌进行博弈，最终达成各自的胜利条件。

### 游戏特色
- **双阵营对抗**：进攻方vs防御方，策略截然不同
- **资源管理**：算力、资金、信息、权限四种资源
- **科技树系统**：通过提升科技等级解锁更强卡牌
- **判定机制**：骰子判定和剪刀石头布判定增加不确定性
- **区域控制**：争夺关键区域获得额外优势

## 快速开始

### 1. 启动游戏
```bash
npm run dev
```
访问 http://localhost:5173/ 开始游戏

### 2. 创建对局
1. 选择游戏模式（1v1 或 2v2）
2. 选择阵营（进攻方/防御方）
3. 选择角色（不同角色有不同技能）
4. 点击"开始游戏"

### 3. 基本操作
- **出牌**：点击手牌，然后点击"出牌"按钮
- **判定**：根据提示进行骰子判定或剪刀石头布
- **结束回合**：点击"结束回合"按钮
- **查看日志**：右侧面板查看游戏记录

## 核心机制详解

### 1. 回合与轮次

#### 回合（Turn）
- 单个玩家的完整行动周期
- 包含7个阶段：判定→恢复→摸牌→行动→响应→弃牌→结束
- 每个玩家在自己的回合内进行操作

#### 轮次（Round）
- 所有玩家各完成一个回合
- 每完成一轮，轮次数+1
- 游戏最多进行12轮

### 2. 资源系统

#### 资源上限
| 资源类型 | 上限值 |
|---------|-------|
| 算力 | 15 |
| 资金 | 20 |
| 信息 | 10 |
| 权限 | 5 |

#### 资源恢复（恢复阶段）
- 算力：恢复至上限
- 资金：恢复至上限
- 信息：恢复至上限
- 权限：不恢复

### 3. 手牌系统

#### 手牌上限
| 轮次范围 | 手牌上限 |
|---------|---------|
| 1-4轮 | 6张 |
| 5-8轮 | 5张 |
| 9-12轮 | 4张 |

#### 摸牌数
- 基础摸牌：2张
- 科技树加成：根据科技等级额外摸牌

### 4. 科技树系统

#### 科技等级
- T0（等级0）：基础卡牌
- T1（等级1）：解锁进阶卡牌
- T2（等级2）：解锁高级卡牌
- T3（等级3）：解锁专家卡牌
- T4（等级4）：解锁大师卡牌

#### 升级条件
- 渗透/安全等级达到阈值自动升级
- 升级后解锁更高科技等级的卡牌

## 卡牌系统

### 卡牌类型

#### 进攻方卡牌
| 类型 | 代码前缀 | 说明 |
|-----|---------|-----|
| 基础侦查 | ATK001-ATK010 | 低消耗，基础渗透 |
| 漏洞利用 | ATK011-ATK020 | 中等消耗，判定类效果 |
| 权限提升 | ATK021-ATK030 | 高消耗，强力渗透 |
| 横向移动 | ATK031-ATK040 | 特殊效果，区域控制 |

#### 防御方卡牌
| 类型 | 代码前缀 | 说明 |
|-----|---------|-----|
| 基础加固 | DEF001-DEF010 | 低消耗，基础安全 |
| 威胁检测 | DEF011-DEF020 | 中等消耗，判定类效果 |
| 访问控制 | DEF021-DEF030 | 高消耗，强力安全 |
| 应急响应 | DEF031-DEF040 | 特殊效果，区域控制 |

### 卡牌效果类型

#### 1. 直接效果
- **渗透提升**：增加进攻方渗透等级
- **安全提升**：增加防御方安全等级
- **资源获取**：获得算力/资金/信息/权限

#### 2. 判定效果
- **骰子判定**：掷1-6点，根据点数决定效果
  - 点数 ≥ 难度：成功
  - 点数 < 难度：失败
  - 掷出6且难度≤3：大成功
  - 掷出1且难度≥4：大失败

- **剪刀石头布**：双方同时出拳
  - 胜利：触发成功效果
  - 失败：触发失败效果
  - 平局：重新判定

#### 3. 持续效果
- **延迟判定**：在目标的下个判定阶段触发
- **光环效果**：持续多回合的效果
- **区域效果**：影响特定区域的效果

### 卡牌使用流程

#### 即时判定卡牌
1. 在行动阶段选择卡牌
2. 系统记录判定事件
3. 进入响应阶段
4. 自动触发判定
5. 根据结果应用效果

#### 延迟判定卡牌
1. 在行动阶段选择卡牌
2. 指定目标玩家
3. 添加延迟判定标记
4. 在目标的判定阶段触发
5. 进行判定并应用效果

## 判定系统

### 骰子判定

#### 判定流程
1. 确定判定难度（1-6）
2. 掷骰子（1-6点）
3. 比较点数与难度
4. 应用对应效果

#### 判定结果
| 结果 | 条件 | 效果 |
|-----|-----|-----|
| 大成功 | 掷出6且难度≤3 | 触发额外强化效果 |
| 成功 | 点数 ≥ 难度 | 触发成功效果 |
| 失败 | 点数 < 难度 | 触发失败效果 |
| 大失败 | 掷出1且难度≥4 | 触发额外负面效果 |

### 剪刀石头布判定

#### 判定流程
1. 双方同时选择出拳
2. 系统比较结果
3. 应用对应效果

#### 胜负规则
- 石头 > 剪刀
- 剪刀 > 布
- 布 > 石头
- 相同则重新判定

## 胜利条件

### 进攻方胜利条件（满足任一）

#### 1. 完全渗透
- 渗透等级达到75
- 持续2个轮次
- 触发"完全渗透"动画

#### 2. 安全瓦解
- 防御方安全等级降至0
- 需要第7轮次后

#### 3. 速攻胜利
- 第6轮次前
- 渗透等级≥50
- 防御方安全等级<50

#### 4. 持久胜利
- 第12轮次结束后
- 渗透等级 > 防御方安全等级

### 防御方胜利条件（满足任一）

#### 1. 绝对安全
- 安全等级达到75
- 触发"绝对安全"动画

#### 2. 威胁清除
- 攻击方渗透等级降至0
- 需要第7轮次后

#### 3. 速防胜利
- 第6轮次前
- 安全等级≥50
- 攻击方渗透等级<50

#### 4. 持久胜利
- 第12轮次结束后
- 安全等级 > 攻击方渗透等级

## 进阶技巧

### 进攻方策略

#### 1. 资源管理
- 优先使用算力和信息类卡牌
- 保留资金用于关键时刻
- 合理使用权限进行特殊行动

#### 2. 判定策略
- 低难度判定（1-2）可以大胆尝试
- 高难度判定（5-6）需要权衡风险
- 利用连击效果叠加收益

#### 3. 时机把握
- 前期快速积累渗透等级
- 中期利用科技优势
- 后期全力冲刺胜利条件

### 防御方策略

#### 1. 资源管理
- 平衡四种资源的使用
- 优先保护安全等级
- 适时反击降低对方渗透

#### 2. 判定策略
- 利用判定失败惩罚对方
- 大成功时最大化收益
- 控制判定节奏

#### 3. 时机把握
- 前期稳固防御
- 中期寻找反击机会
- 后期确保胜利条件

### 通用技巧

#### 1. 卡牌组合
- 相同类型卡牌连击获得加成
- 低费卡牌快速过牌
- 高费卡牌关键时刻使用

#### 2. 日志分析
- 查看对手资源变化
- 分析对手手牌类型
- 预测对手策略

#### 3. 科技树规划
- 根据策略选择升级方向
- 提前规划需要的卡牌
- 利用科技优势压制对手

## 常见问题

### Q1: 为什么我的响应阶段没有自动跳过？
A: 如果你在行动阶段使用了即时判定卡牌，系统会在响应阶段进行判定，不会跳过。

### Q2: 如何查看卡牌详情？
A: 鼠标悬停在卡牌上3秒或右键点击卡牌，可以查看完整详情。

### Q3: 延迟判定和即时判定有什么区别？
A: 
- **即时判定**：在你的响应阶段立即触发判定
- **延迟判定**：在目标玩家的下个判定阶段触发

### Q4: 科技等级如何提升？
A: 渗透/安全等级达到阈值时自动提升，解锁更高等级的卡牌。

### Q5: 如何获得区域控制加成？
A: 在区域放置标志物，当己方标志物数量超过对方时获得控制权和加成。

### Q6: 连击效果如何触发？
A: 连续使用相同类型的卡牌，会触发连击效果获得额外加成。

### Q7: 游戏日志在哪里查看？
A: 游戏界面右侧是日志面板，可以查看每轮的详细记录，支持翻页查看。

### Q8: 为什么有时候不能出牌？
A: 可能原因：
- 资源不足
- 不是行动阶段
- 手牌已达上限
- 受到某些效果限制
