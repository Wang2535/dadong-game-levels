## 项目概述

### 项目目标
基于"大东话安全"的81篇网络安全文章，开发一个教育性的闯关模式，将专业知识转化为游戏关卡，让玩家在游戏中学习网络安全知识。

### 核心价值
- **教育性**：将专业知识转化为可玩内容
- **趣味性**：通过游戏机制增强学习动力
- **渐进性**：由浅入深，循序渐进
- **实践性**：理论知识与实际操作结合

### 目标用户
- 网络安全初学者
- 企业安全培训人员
- 对网络安全感兴趣的游戏玩家
- 高校网络安全课程学生

## 81关关卡设计

### 关卡设计原则

#### 1. 知识递进
- 每3关为一个知识单元
- 第1关：概念介绍
- 第2关：原理讲解
- 第3关：实战应用

#### 2. 难度曲线
```
难度
  ↑
10│                                    ★ Boss关
  │                              ★
 8│                        ★
  │                  ★
 6│            ★
  │      ★
 4│ ★
  │
 2│
  └────────────────────────────────────→ 关卡
    1  9  18  27  36  45  54  63  72  81
```

#### 3. 游戏机制融合
- 将知识点转化为卡牌效果
- 使用判定机制模拟攻击/防御成功率
- 通过资源管理体现安全投入

## 技术架构规划

### 1. 数据模型扩展

#### 关卡数据模型
```typescript
interface Level {
  id: number;                    // 关卡ID (1-81)
  moduleId: number;              // 所属模块 (1-9)
  title: string;                 // 关卡标题
  description: string;           // 关卡描述
  knowledgePoints: string[];     // 知识点列表
  articleRef: string;            // 对应文章引用
  
  // 游戏配置
  gameMode: 'tutorial' | 'battle' | 'puzzle' | 'boss';
  difficulty: number;            // 难度等级 (1-10)
  timeLimit?: number;            // 时间限制（秒）
  
  // 卡牌配置
  availableCards: string[];      // 可用卡牌列表
  startingHand: string[];        // 初始手牌
  
  // 特殊规则
  specialRules?: SpecialRule[];  // 特殊规则
  
  // 胜利条件
  victoryConditions: VictoryCondition[];
  
  // 奖励
  rewards: {
    xp: number;
    unlockCards?: string[];
    unlockCharacters?: string[];
  };
}
```

#### 玩家进度模型
```typescript
interface PlayerProgress {
  playerId: string;
  completedLevels: number[];     // 已完成关卡
  currentLevel: number;          // 当前关卡
  starsPerLevel: Map<number, number>; // 每关获得的星星数
  unlockedCards: string[];       // 已解锁卡牌
  achievements: string[];        // 已获得成就
  
  // 学习数据
  knowledgeMastery: Map<string, number>; // 知识点掌握度
  quizScores: Map<number, number>;       // 测验分数
}
```

### 2. 关卡类型设计

#### 教程关 (Tutorial)
- 引导式教学
- 强制步骤提示
- 知识点讲解弹窗
- 无失败惩罚

#### 对战关 (Battle)
- 与AI对手对战
- 特定胜利条件
- 资源/卡牌限制
- 星级评价系统

#### 解谜关 (Puzzle)
- 特定场景解谜
- 最优解挑战
- 步骤限制
- 提示系统

#### Boss关 (Boss)
- 综合考验
- 多阶段战斗
- 特殊机制
- 丰厚奖励

### 3. 教育系统集成

#### 知识点讲解系统
```typescript
interface KnowledgePoint {
  id: string;
  title: string;
  content: string;           // 讲解内容
  relatedArticle: string;    // 关联文章
  quizQuestions: Quiz[];     // 测验题目
  
  // 多媒体支持
  images?: string[];
  videos?: string[];
  interactiveDemo?: string;  // 交互式演示
}
```

#### 测验系统
```typescript
interface Quiz {
  id: string;
  question: string;
  options: string[];
  correctAnswer: number;
  explanation: string;       // 答案解析
  difficulty: number;
}
```
