## 一、设计概述与视觉风格

### A1.1 设计理念

#### `A1.1.1` **核心视觉主题**
> **"赛博朋克 × 网络安全 × 东方美学"**

融合未来科技感与传统东方元素，营造数字世界的博弈氛围。

#### `A1.1.2` **视觉关键词**

| 关键词 | 视觉表现 | 应用场景 |
|--------|---------|---------|
| **科技感** | 电路纹理、数据流、全息投影 | 卡牌边框、UI元素 |
| **博弈感** | 对称构图、对比色彩、动态线条 | 角色立绘、战斗场景 |
| **神秘感** | 暗色调、霓虹光效、几何图形 | 背景设计、特效表现 |
| **东方韵** | 水墨元素、书法字体、传统纹样 | 角色服饰、装饰元素 |

### A1.2 艺术风格定位

#### `A1.2.1` **整体风格参考**

- **赛博朋克**: 《赛博朋克2077》《攻壳机动队》
- **卡牌游戏**: 《炉石传说》《杀戮尖塔》
- **东方美学**: 《影之刃》《阴阳师》
- **科技感UI**: 《看门狗》《Deus Ex》

#### `A1.2.2` **风格统一原则**

1. **色彩统一**: 严格使用定义的色彩系统，禁止随意添加新颜色
2. **线条统一**: 主要使用2px描边，圆角统一为4px或8px
3. **质感统一**: 使用玻璃态（Glassmorphism）和霓虹光效为主
4. **层次统一**: 遵循"背景→中景→前景→UI"的四层结构

### A2.2 色彩应用规范

#### `A2.2.1` **渐变规范**

**主渐变**:
```css

/* 防御方渐变 */
```

```css
/* 红色霓虹 */

/* 蓝色霓虹 */

/* 金色霓虹（传说卡） */

```

| 应用场景 | 透明度 | 说明 |

| 悬浮提示 | 70-80% | 半透明 |
| 禁用状态 | 40-50% | 明显透明 |

## 九、特效与动画规范

### A9.1 卡牌特效

#### `A9.1.1` **抽卡动画**

**动画序列**:
1. 卡牌从牌库飞出（0.3s，ease-out）
2. 卡牌翻转显示正面（0.2s，ease-in-out）
3. 卡牌飞入手牌区（0.3s，ease-out）
4. 手牌重新排列（0.2s，ease-out）

**特效**:
- 稀有卡: 蓝色光效拖尾
- 史诗卡: 紫色光效拖尾 + 粒子
- 传说卡: 金色光效拖尾 + 大量粒子

#### `A9.1.2` **打出动画**

**动画序列**:
1. 卡牌从手牌区放大（0.2s，ease-out）
2. 卡牌飞入战场中央（0.4s，ease-in-out）
3. 卡牌效果触发（0.3s）
4. 卡牌进入弃牌堆或战场（0.2s）

#### `A9.1.3` **连击特效**

**触发条件**: 连击卡效果触发

**特效描述**:
- 卡牌边缘金光渐变闪烁
- 频率: 1秒1闪
- 持续时间: 直到连击效果结算
- 颜色: 金色 #FFD700 → 白色 #FFFFFF → 金色 #FFD700

### A9.2 判定特效

#### `A9.2.1` **骰子判定**

**动画序列**:
1. 骰子出现并旋转（1s，ease-out）
2. 骰子停止，显示结果（0.2s）
3. 结果判定，显示成功/失败（0.3s）

**成功特效**:
- 绿色光效爆发
- 数字放大闪烁
- 成功音效

**失败特效**:
- 红色光效暗淡
- 数字缩小
- 失败音效

#### `A9.2.2` **猜拳判定**

**动画序列**:
1. 双方选择暗牌（2s倒计时）
2. 同时亮牌（0.3s）
3. 判定胜负（0.5s）
4. 触发效果（0.5s）

**胜负特效**:
- 胜利方: 光效增强，角色动作
- 失败方: 光效暗淡，角色受击动作
- 平局: 双方光效平衡，特殊音效

### A9.3 胜利/失败特效

#### `A9.3.1` **进攻方胜利**

**特效描述**:
- 屏幕红色光效笼罩
- 渗透等级数字快速上升动画
- "完全渗透"文字出现
- 防御方系统崩溃动画
- 胜利音乐 + 欢呼音效

#### `A9.3.2` **防御方胜利**

**特效描述**:
- 屏幕蓝色光效笼罩
- 安全等级数字快速上升动画
- "绝对安全"文字出现
- 进攻方被击退动画
- 胜利音乐 + 警报解除音效

### A9.4 动画时间规范

| 动画类型 | 持续时间 | 缓动函数 | 备注 |
|---------|---------|---------|------|
| **快速反馈** | 0.1-0.2s | ease-out | 按钮点击 |
| **标准动画** | 0.3-0.5s | ease-in-out | 卡牌移动 |
| **复杂动画** | 0.8-1.2s | ease-out | 判定动画 |
| **强调动画** | 1.5-2.0s | ease-in-out | 胜利动画 |
