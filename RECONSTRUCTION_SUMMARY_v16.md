## 核心系统设计

### 1. 等级变化公式

```typescript
最终变化 = 基础值 + 科技加成 + 连击加成 + 判定修正

其中：
- 科技加成 = floor(科技等级 × 0.5)
- 连击加成 = min(连击次数 × 0.5, 3)
- 判定修正 = 大成功(+2) / 大失败(-1) / 其他(0)
```

### 2. 科技树系统

| 科技等级 | 所需等级 | 解锁卡牌类型 | 效果 |
|---------|---------|-------------|------|
| T1 | 0 | 基础侦查/防御 | 无 |
| T2 | 15 | 漏洞利用/入侵检测 | 判定难度-1 |
| T3 | 30 | 权限提升/主动防御 | 判定难度-1，等级+1 |
| T4 | 45 | 高级攻击/纵深防御 | 判定难度-1，等级+1 |
| T5 | 60 | 完全控制/绝对安全 | 判定难度-1，等级+2 |

### 3. 卡牌效果类型

| 效果类型 | 攻击方 | 防御方 | 说明 |
|---------|--------|--------|------|
| security_reduce | ✅ | ❌ | 降低对方安全等级 |
| infiltration_gain | ✅ | ❌ | 提升自身渗透等级 |
| security_gain | ❌ | ✅ | 提升自身安全等级 |
| infiltration_reduce | ❌ | ✅ | 降低对方渗透等级 |
| resource_gain | ✅ | ✅ | 获得资源 |
| draw | ✅ | ✅ | 抽牌 |
| protection | ✅ | ✅ | 保护效果 |

### 导入新系统

```typescript
// 导入v16类型
import { Card, Player, GameState } from './types/index_v16';

// 导入v16数据
import { ATTACK_CARDS, DEFENSE_CARDS } from './data/index_v16';
import { calculateTechLevel, getTechLevelEffects } from './data/techTree';

// 导入v16引擎

```

### 使用等级计算器

```typescript
import { executeLevelCalculation } from './engine/LevelCalculator';

const result = executeLevelCalculation(
  30,        // 当前等级
  3,         // 基础变化值
  2,         // 科技等级
  2,         // 连击次数
  true,      // 是否增益
  diceResult // 判定结果（可选）
);

console.log(result.totalChange); // 总变化值
console.log(result.result.newValue); // 新等级值
```

### 使用效果引擎

```typescript
import { executeCardEffects } from './engine/EffectEngine';

const results = executeCardEffects(card, {
  gameState,
  sourcePlayer,
  targetPlayer,
  sourceCard: card,
  diceResult,
  consecutiveUses: 2
});
```
